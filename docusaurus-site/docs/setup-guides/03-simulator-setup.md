## Limitations

| Limitation | Workaround |
|-----------|-----------|
| **No real hardware** | Upgrade to Arduino/Raspberry Pi later |
| **Limited realism** | Simulates ideal conditions only |
| **No physical interaction** | Can't control actual motors/servos |

---

## When to Use Simulator vs Hardware

**Start with Simulator**:
- Learning concepts
- Testing code logic
- First-time programming
- Teaching in classroom

**Upgrade to Hardware**:
- Need real-world behavior
- Want to control physical devices
- Building actual projects
- Advanced experiments
